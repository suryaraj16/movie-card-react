# Movie Card React Component

A reusable MovieCard component built using React + TypeScript.

## Features
- Reusable component structure
- Strongly typed props (TypeScript)
- Responsive grid layout
- Clean folder architecture
- Smooth hover animation effects

##  Tech Stack
- React
- TypeScript
- Vite
- CSS

## 📂 Project Structure
src/
 ├── components/
 │     ├── MovieCard.tsx
 │     └── MovieCard.css
 ├── App.tsx
 ├── App.css
 ├── main.tsx
